# A hkl guesser for FCC
Give out possible hkl values for an fcc diffraction pattern, with the length of two bases and the angle between as input them, and also filtering on a specified zone axis.


This code was made for the compulsotory TEM report in TFY4220 Faste stoffers fysikk (2021 VÅR). The student group who made this report was Petter Grønlie, Brynjar Morka Mæhlum, Andreas Rosnes and Øyvind Brønstad Rørbakken.
